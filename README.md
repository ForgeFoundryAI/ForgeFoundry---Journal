# ForgeFoundry

**Building the machine while using the machine.**

A transparent journey of a non-technical founder learning to build with AI — from zero coding experience to creating an autonomous AI system that helps solopreneurs discover, evolve, and launch real micro SaaS products.

This repository showcases not only the product, but also the skills and talents developed along the way.

---

## About This Project

ForgeFoundry scans real conversations in communities, identifies genuine pain points, scores them honestly (0–100.0), and helps users turn those sparks into buildable, revenue-generating products.

At its core is **Crucible** — the hidden "rocket engine" that powers discovery, evolution, and completion of ideas.

---

## Current Status (April 2026)

- **Tiers**: Spark (Free) → Forge ($39/mo) → Foundry ($99/mo)
- **Flagship Products in Progress**: Temper (Polymarket companion), BidGuard (contractor tool)
- **Crucible v9**: Autonomous engine with strong Evolution Layer + Feedback Assistant
- **Philosophy**: Use AI as a powerful assistant, not a replacement for human effort.

## Tech Stack (Evolving)

**Current Foundation**
- AI: Grok (xAI)
- No-code / Rapid development: Bubble.io, n8n, Supabase
- Database & Auth: Supabase
- Hosting: Vercel / Bubble

**Expanding Into (As We Build)**
- Python (for advanced backend logic, automation scripts, and custom agents)
- Browser Extensions (JavaScript/TypeScript)
- React / Next.js (for more complex frontend components)
- Additional tools as needed (FastAPI, Docker, etc.)

This repository will document the growth of the tech stack as we move from no-code foundations to more robust, production-grade solutions.

## The Goal

Hitting a home run on your first at-bat in the major leagues is one of the rarest achievements in baseball — only **138 players** in MLB history have ever done it.

My goal with ForgeFoundry is ambitious in the same spirit: to build something meaningful on the first major swing, while openly documenting the entire process. Whether this becomes a grand slam or a solid base hit, the journey itself will showcase the skills, persistence, and learning that come from treating AI as a true collaborator.
